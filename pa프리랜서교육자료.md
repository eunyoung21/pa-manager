# 📘 PA Manager 사용 가이드 (프리랜서용)

인플루언서 협업을 **리스트업 → 컨택 → 계약 → 출고 → 영상검수**까지 한 곳에서 관리하는 프로그램이에요.
아래 순서대로 따라 하면 됩니다. (모든 입력은 **자동 저장**돼요!)

---
0. 접속 & 로그인
1. 주소: **https://pa-manager.onrender.com**
2. 받은 **아이디 / 비밀번호**로 로그인
3. 로그인하면 위에 `👷 프리랜서 모드`라고 떠요. (정상이에요)

> 💡 **데이터를 수정한 뒤에는 새로고침(F5)** 한 번 해주세요. 다른 사람이 바꾼 최신 내용을 받아오기 위해서예요.

---

## 1. 📋 STEP1 리스트업 — 좋은 인플루언서 찾아 등록
1. 왼쪽 위 **＋ 새 인플루언서 등록** (또는 STEP1 탭 → ＋ 인플루언서 추가)

2. 입력 항목
   - **채널명 / 채널 링크** (인스타 계정 또는 URL)
   - **팔로워 수**
   - **페르소나/카테고리** (예: 낚시·아웃도어 / 40대 남성)
   - **특이사항** 
   - **담당 PA** = 본인 이름 선택

3. **검수 상태는 그대로 두면 자동으로 "검수대기"** 예요. (프리랜서는 승인/반려 못 바꿔요 — 담당자가 확인합니다)

4. 셀을 클릭하면 바로 수정돼요. **Enter = 저장, Esc = 취소**

> 담당자가 **승인**하면 자동으로 **STEP2 컨택현황**에 추가됩니다.

---

## 2. 📨 STEP2 컨택현황 — 컨택 진행 + 수당
승인된 인플루언서가 자동으로 나타나요. 여기서 진행 상황을 업데이트합니다.

- **진행 상태** (클릭해서 변경, 색으로 구분)
  - 🟠 컨택 전 → 🔵 진행중 → 🟢 컨택 완료 → 🟣 계약서 작성 → 🔴 거절
  - **거절**을 누르면 **진행 중 목록에서 빠지고**, 🚫 거절 / 📦 보관 칩에서 모아 볼 수 있어요.
- **DM 발송** 버튼을 **Y**로 → 💰 **2,000원** 자동 적립
- **협업 성사** 버튼을 **Y**로 → 💰 **20,000원** 추가 적립 + 성사일 자동 기록
- **단가**: 협의된 단가가 **단가 칸에 표시**돼요. (확인용 — 이제 프리랜서도 볼 수 있어요)
- 제품 발송일·예상 게시일·출고·계약서 여부도 업데이트

> ⭐ **DM 보냈으면 꼭 DM발송 Y, 협업 성사되면 꼭 협업성사 Y!** 이게 정산(수당)의 기준이에요.

### 📦 목록이 깔끔해졌어요 (진행 중 / 보관)
- 기본 화면은 **진행 중**인 인플루언서만 보여줘요. 거절했거나 마무리된 분은 자동으로 빠져서 가독성이 좋아져요.
- 상단 **📦 보관** 칩을 누르면 거절·마무리된 분을 모아 볼 수 있어요.
- **데이터·수당 정산은 그대로 보존**돼요(지워지는 게 아니에요). 검색창은 보관 항목까지 포함해서 누구든 찾을 수 있어요.

### 개인정보 제출
- 협업 성사된 분은 컨택현황 -> 개인정보** 칸의 **📝 제출** 버튼으로 통장사본·신분증·주소 등을 입력
- 이미 제출했으면 **✅ 제출됨·수정** 을 눌러 **언제든 다시 수정** 가능

> 🔒 **개인정보 보호:** 관리자가 인플루언서 단가를 지급(정산)하면, **3일 뒤 그 인플루언서의 개인정보는 자동으로 폐기**돼요. (정산/폐기는 관리자가 처리합니다)

---

## 3. 📦 출고 — 제품 발송 신청
1. 출고 탭 → **＋ 출고 추가**
2. 요청자(본인)·상품명·수령자명·연락처·배송지 입력
3. 담당자가 확인 후 송장번호 입력 & 완료 처리

---

## 4. 🎬 영상검수
- 업로드된 영상/결과를 확인하고 상태를 업데이트하는 탭이에요.

## 5. 💬 DM 템플릿
- 컨택할 때 쓸 **DM 문구 모음**이에요. 마음에 드는 걸 복사해서 보내면 돼요.
- 난이도: 하(즉시 발송) / 중(검토 후) / 상(톤 보고 수정)

---

## ✅ 꼭 기억할 것
1. **DM 보내면 DM발송 Y, 성사되면 협업성사 Y** (수당 누락 방지)
2. 담당 PA에 **본인 이름** 꼭 선택
3. 데이터 수정 후 **새로고침(F5)**
4. 입력은 **자동 저장** — 따로 저장 버튼 없어요
5. 막히면 담당자(관리자)에게 문의 🙂
